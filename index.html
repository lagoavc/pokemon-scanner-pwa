📷 Como usar

  1. Aponte a câmara para uma carta Pokémon
  2. Prima o botão de captura (círculo branco)
  3. A imagem é enviada para OCR.space que tenta extrair o nome e número
  4. Confirme ou corrija os dados manualmente
  5. Prima "Pesquisar" para encontrar a carta na Pokémon TCG API
  6. Defina estado, idioma, preço e quantidade
  7. Prima "Adicionar à Lista"

🔑 API Key do OCR.space

O OCR.space é o serviço que lê o texto das imagens. Precisa de uma chave gratuita:

  1. Abra ocr.space/ocrapi
  2. Registe-se (email + palavra-passe)
  3. A chave API será enviada para o seu email
  4. Copie a chave e cole abaixo

A chave fica guardada apenas neste telemóvel (localStorage).

⚠ Limitações

  • Plano gratuito OCR.space: 10 requests/dia. Se exceder, a app avisa.
  • Qualidade do OCR: cartas com texto dourado, foil ou brilho podem ter leitura incorreta.
  • Pokémon TCG API: gratuita mas com limite de pedidos. Pode falhar em hora de pico.
  • Conexão: requer internet para OCR e pesquisa de cartas.

📈 Backup / Restauro

Use Exportar Backup para guardar toda a coleção + chave API do OCR.space num ficheiro JSON. Depois pode Importar Backup noutro telemóvel para restaurar tudo.

🕵 Privacidade

As fotos que tira são enviadas para:

  • OCR.space — para reconhecimento de texto
  • Pokémon TCG API — para pesquisar detalhes das cartas

Nenhuma imagem ou dado é guardado em servidores nossos. A sua API key do OCR.space fica apenas no seu dispositivo.

O PokéScanner utiliza serviços externos para funcionar:

  • OCR.space — as fotos das cartas são enviadas para reconhecimento de texto
